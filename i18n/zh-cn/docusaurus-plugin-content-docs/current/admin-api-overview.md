Pulsar 管理 API 使您能够以编程方式管理集群。例如，您可以创建、更新、删除和管理 Pulsar 实例中的所有实体（如集群、命名空间、租户、Topic、Schema、Connector、Function 等），并为数据、资源和安全设置各种策略。
