启动和运行 Pulsar 非常简单。下载、安装并试用它。

你有三种选择。点击任意链接开始你的 Pulsar 之旅！

* [在本地运行独立的 Pulsar 集群](getting-started-standalone.md) - 在单台机器上以独立模式运行单个 Pulsar 实例。
* [在 Docker 中运行独立的 Pulsar 集群](getting-started-docker.md) - 在 Docker 容器中运行一个或多个 Pulsar 实例。如果你想快速在本地笔记本电脑上设置一个小型 Pulsar 集群，请参阅[使用 Docker Compose 运行 Pulsar 集群](getting-started-docker-compose.md)。
* [在 Kubernetes 中运行独立的 Pulsar 集群](getting-started-helm.md) - 使用 Helm chart 在 Kubernetes 中运行一个或多个 Pulsar 实例。
